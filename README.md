# Banking Management System

## Overview

This project is a Python-based Banking Management System designed to simulate core banking operations such as managing customers, accounts, transactions, loans, and cards.

The goal of this project is to apply object-oriented programming (OOP) principles to build a modular and scalable system.

---

## Features

* Customer management
* Account creation (Savings & Checking)
* Deposit, withdrawal, and fund transfer
* Transaction tracking
* Loan management
* Card system (Debit/Credit)
* Authentication (Admin & Customer roles)
